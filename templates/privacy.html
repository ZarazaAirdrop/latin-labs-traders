Política de Privacidad

Última actualización: Junio 2026

1. Información que Recopilamos

No recopilamos información personal identificable a menos que nos la proporciones voluntariamente. Podemos recopilar datos de uso anónimos a través de Google Analytics y Facebook Pixel para mejorar nuestros servicios.

2. Google Analytics

Utilizamos Google Analytics para analizar el tráfico del sitio. Google Analytics recopila información como tu dirección IP (anonimizada), navegador, sistema operativo y páginas visitadas. Puedes obtener más información en la Política de Privacidad de Google.

3. Facebook Pixel

Utilizamos Facebook Pixel para medir la efectividad de nuestras campañas. Facebook puede utilizar cookies para rastrear tu actividad. Consulta la Política de Privacidad de Facebook para más información.

4. Cookies

Este sitio utiliza cookies necesarias para el funcionamiento básico y cookies de terceros (Google Analytics, Facebook Pixel) para análisis y marketing. Puedes configurar tu navegador para rechazar cookies, aunque algunas funcionalidades pueden verse afectadas.

5. API de OpenRouter

Cuando utilizas el asistente de IA, tu consulta se envía a la API de OpenRouter para su procesamiento mediante el modelo Mistral Small 24B. No almacenamos ni compartimos el contenido de tus conversaciones con terceros. Consulta la Política de Privacidad de OpenRouter.

6. Almacenamiento de Datos

No almacenamos datos de usuarios en servidores propios. No hay registro de cuentas, contraseñas ni información financiera. Todo el procesamiento ocurre en tu navegador o a través de APIs externas.

7. Tus Derechos

Tienes derecho a solicitar información sobre los datos que procesamos. Para ejercer tus derechos, contáctanos en latinlabstraders@gmail.com.

8. Cambios a esta Política

Podemos actualizar esta política periódicamente. Los cambios serán publicados en esta página con la fecha de última actualización.
